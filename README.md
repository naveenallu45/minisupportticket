# Mini Ticket Management

A small ticket management app built with Next.js, MongoDB, NextAuth, and Socket.IO.

## Getting Started

Install dependencies:

```bash
npm install
```

Create a `.env` file with the required database and auth values, then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
